* Installation des dépendances :
  * Via **Yarn** : `yarn`
  * Ou, via **NPM** : `npm install`


## :punch: Utilisation
* Exécuter Brunch en mode **watch** :
  * Via **Yarn** : `yarn start`
  * Ou, via **NPM** : `npm start`
